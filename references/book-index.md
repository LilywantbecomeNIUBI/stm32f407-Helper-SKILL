# 教材索引

把当前任务快速映射到教材章节。先检查当前工程和本地例程，再用本文件定位教材，不要上来通读整本书。

如果用户教材的章节名与这里不完全一致，按“功能类别”匹配，不要机械依赖章节号。

## 使用原则

按以下顺序使用资料：

1. 当前工程已有代码和 `.ioc`
2. 本地例程
3. 本文件定位到的教材章节
4. 引脚表 / 板载资源表 / 原理图

教材主要用于：

- 理解模块原理和初始化顺序
- 抽取最小可迁移驱动逻辑
- 辅助判断时序、接口、联调方法
- 在例程较重时帮助裁剪代码

## 基础章节

- 第 7 章，认识 HAL 库
  适用于理解 HAL 风格、驱动接口、回调关系。
- 第 10 章，STM32CubeMX 简介
  适用于规划外设、生成初始化代码、理解工程骨架。
- 第 11 章，时钟系统
  适用于主频、PLL、SYSCLK、APB 分频、定时器时钟、串口波特率、ADC 时钟等问题。
- 第 12 章，SYSTEM 文件夹
  适用于理解例程里的 `delay`、`sys`、`usart` 等公共支持代码，但不要把工程壳整体照搬。

## 模块映射

### GPIO / KEY / BEEP / EXTI

- 第 13 章，跑马灯实验
  用于 GPIO 输出、LED、普通输出控制。
- 第 14 章，蜂鸣器实验

- 第 15 章，按键输入实验

- 第 16 章，外部中断实验
  用于 EXTI、NVIC、按键中断和边沿触发。

### USART

- 第 17 章，串口通信实验
  用于串口收发、`printf` 重定向、中断接收、基础通信流程。

### TIM / PWM / 输入捕获 / 编码器

- 第 20 章，基本定时器实验

- 第 21 章，通用定时器实验

- 第 22 章，高级定时器实验

### OLED / TFT

- 第 24 章，OLED 显示实验

### DMA / ADC

- 第 32 章，内部温度传感器实验

- 第 33 章，光敏传感器实验

### I2C / SPI / CAN

- 第 36 章，IIC 实验

- 第 39 章，CAN 通信实验
  用于 CAN 总线。

### 常见传感器

## 特别提醒

- 例程和教材优先用于迁移“模块逻辑”和“联调方法”，不是直接搬工程结构。
- 当教材示例是 Keil/MDK 工程时，保留逻辑、时序、接口和 HAL 用法，不保留工程壳。
- 若当前任务涉及时钟、DMA、NVIC、引脚复用，回到 `CubeMX` 和时钟章节辅助判断，但最终仍以当前 `.ioc` 为准。
